# TP N°01 POO - Programmation des structures algorithmiques en Java

Ce dépôt contient le rapport interactif et les codes sources du premier Travail Pratique (TP) du module de Programmation Orientée Objet (POO) dispensé à l'USTHB (Faculté de Génie Electrique, Département d'Automatique, Spécialité M1 AII).

## ✨ Nouveautés de cette version
* **Interface "Google Pixel" :** Design moderne et épuré utilisant les couleurs officielles Google (Pixel Blue, Cyan) et la typographie `Google Sans`.
* **Terminal Interactif en Temps Réel :** Plus besoin de compiler le code pour tester les résultats ! Le rapport intègre un terminal simulé en JavaScript permettant d'exécuter la logique des exercices Java directement depuis le navigateur.

## 📋 Contenu des Exercices

1. **Exercice 1 : Test de majorité**
   - *Objectif :* Utiliser la structure conditionnelle `if/else` pour déterminer si un utilisateur est majeur ou mineur en fonction de son âge.
2. **Exercice 2 : Message de bienvenue**
   - *Objectif :* Manipuler les chaînes de caractères (`String`) et la concaténation pour afficher un message personnalisé.
3. **Exercice 3 : Parité d'un nombre**
   - *Objectif :* Utiliser l'opérateur modulo (`%`) pour vérifier si un entier saisi est pair ou impair.
4. **Exercice 4 : Détection de nombre premier**
   - *Objectif :* Implémenter une boucle `for` et optimiser la recherche de diviseurs (jusqu'à la racine carrée) pour déterminer si un nombre est premier.

## 🚀 Comment utiliser ce projet ?

### 1. Tester directement dans le navigateur (Recommandé)
Il vous suffit de double-cliquer sur le fichier `TP01_POO.html` pour l'ouvrir dans n'importe quel navigateur web moderne (Chrome, Edge, Safari). 
Naviguez vers la section de votre choix et utilisez la fenêtre de terminal noire intégrée pour saisir vos valeurs et cliquer sur **Exécuter**. Le résultat s'affichera instantanément.

### 2. Exécuter les codes Java (Méthode classique)
Les codes sources Java complets sont fournis dans chaque section. Pour les exécuter sur votre machine :
1. Assurez-vous d'avoir le **JDK (Java Development Kit)** installé.
2. Copiez le code source d'un exercice et collez-le dans un fichier portant le nom de la classe (ex: `Exercice1.java`).
3. Ouvrez un terminal système et compilez-le :
   ```bash
   javac Exercice1.java
