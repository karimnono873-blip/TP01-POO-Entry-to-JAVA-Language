1. Objectifs et Outils du TP

Le but de ce TP est de faire la programmation des algorithmes et des structures algorithmiques de contrôle par l'utilisation du langage Java (POO).

L'énoncé composé groupe des actions qui sont ensuite considérées comme une seule action. Dans notre notation algorithmique, les énoncés à composer seront placés entre les deux parenthéseurs début et fin. Les parenthéseurs sont représentés par les accolades ouvrantes et fermantes en Java.

En JAVA, il n'y a pas à proprement parler de séparateur d'énoncé. Toutefois, un point-virgule est nécessaire pour terminer un énoncé simple.

2. Rappels Théoriques

2.1 Déclaration des variables en JAVA

Les types entiers byte, short, int, long sont signés et représentés en complément à 2. Ils se distinguent par leur cardinal. Les entiers du type byte sont représentés sur 8 bits, ceux du type short sur 16 bits, ceux du type int sur 32 bits et ceux du type long sur 64 bits.

2.2 Autres Types

Les types réels : Java définit plusieurs types réels: float et double. Les réels du type float sont en simple précision codés sur 32 bits, ceux du type double sont en double précision codés sur 64 bits. Exemple : float distance, rayon;

Le type booléen : Le type boolean définit les deux valeurs true et false. Exemple : boolean present, onContinue;

3. Exercice 1

Ecrire un programme en java qui permet de tester l'âge d'une personne saisi à l'écran, il lui affiche mineur si l'âge est inférieur à 18 ans et majeur sinon.

Exercice1.java
import java.util.Scanner;

public class Exercice1 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Veuillez saisir votre age : ");
        int age = sc.nextInt();
        
        if (age < 18) {
            System.out.println("mineur");
        } else {
            System.out.println("majeur");
        }
        sc.close();
    }
}

5. Exercice 3

Écrire un programme Java qui lit sur l'entrée un entier N, et qu'il affiche un message à la fin du programme, que ce nombre N est pair ou impair.

Exercice3.java
import java.util.Scanner;

public class Exercice3 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Saisissez un nombre entier N : ");
        int n = sc.nextInt();
        
        if (n % 2 == 0) {
            System.out.println("Le nombre " + n + " est pair.");
        } else {
            System.out.println("Le nombre " + n + " est impair.");
        }
        sc.close();
    }
}

6. Exercice 4

Ecrire un programme Java qui demande à l'utilisateur de saisir un nombre entier n et de lui afficher si le nombre tapé est premier ou non ?

Exercice4.java
import java.util.Scanner;

public class Exercice4 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Veuillez saisir un nombre n : ");
        int n = sc.nextInt();
        boolean estPremier = true;
        
        if (n <= 1) {
            estPremier = false;
        } else {
            for (int i = 2; i <= Math.sqrt(n); i++) {
                if (n % i == 0) {
                    estPremier = false;
                    break;
                }
            }
        }
        
        if (estPremier) {
            System.out.println(n + " est premier.");
        } else {
            System.out.println(n + " n'est pas premier.");
        }
        sc.close();
    }
}
